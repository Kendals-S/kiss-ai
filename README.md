# kiss-ai

Spring Boot 4.1.0 multi-module project.

## Module

- `kiss-ai`: Spring Boot application module

## Coordinates

- Group: `com.ks`
- Artifact: `kiss-ai`
- Version: `1.0.0-SNAPSHOT`
- Java: `21`

## Run

```bash
mvn -pl kiss-ai spring-boot:run
```

Or build from the parent project:

```bash
mvn clean package
java -jar kiss-ai/target/kiss-ai-1.0.0-SNAPSHOT.jar
```
